# Интерактивная цветовая доска


Интерактивная веб‑страница с сеткой квадратов, которые меняют цвет и эффект свечения при наведении курсора.  
При уходе курсора элемент возвращается к исходному состоянию.  


## Функционал


* **Сетка из 500 квадратов** создаётся динамически через JavaScript.
* **Случайный цвет при наведении:** каждый квадрат получает случайный цвет из предопределённого массива.
* **Эффект свечения:** при наведении добавляется `box‑shadow` для визуального эффекта.
* **Возврат к исходному виду:** при уходе курсора квадрат возвращается к базовому цвету и тени.
* **Лёгкость настройки:** легко изменить количество квадратов, цвета и стили.
